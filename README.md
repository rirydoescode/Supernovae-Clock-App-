# ⏰ JavaFX Clock Suite

A desktop clock application built with JavaFX, featuring three core tools in one clean tabbed interface — Clock, Stopwatch, and Timer — wrapped in playful, hand-animated visuals.

## ✨ Features

- **Clock** — real-time animated clock display
- **Stopwatch** — precision time tracking
- **Timer** — countdown timer with custom durations
- **Custom animations** — including an animated star, a ghost character, and a pixel-art supernova effect, all built from scratch in JavaFX

## 🛠️ Tech Stack

- Java
- JavaFX

## 🎨 Design Notes

This project was built with a strong focus on creative direction — every animation (the star, the ghost, the supernova) was custom-designed rather than pulled from a library, aiming for a playful, distinct visual identity rather than a generic utility-app look.

## 📌 Status

Originally built as a desktop app. A mobile (Flutter) port is planned.
